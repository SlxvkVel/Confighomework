# Вариант №7

Инструмент командной строки для преобразования текста из учебного конфигурационного языка в формат TOML. Программа разработана на языке Go и обрабатывает все конструкции учебного языка согласно моему варианту.

## Описание всех функций и настроек

### Синтаксис учебного конфигурационного языка

#### 1. Многострочные комментарии
```conf
{-
Это многострочный
комментарий
-}
```

#### 2. Числа
- Восьмеричные: `0o755`, `0O644`
- Десятичные: `8080`, `42`
- 
#### 3. Массивы
```conf
HOSTS := [ значение, значение, значение, ... ];
```

#### 4. Имена переменных
- Только заглавные буквы и подчеркивания: `[A-Z]+`

#### 5. Строки
```conf
HOST := @"Это строка";
```

#### 6. Объявление констант
```conf
имя := значение;
```

#### 7. Константные выражения (префиксная форма)
```conf
SUM := ?(+ 10 20);
```

#### Поддерживаемые операции:
- `+` - сложение
- `-` - вычитание  
- `ord()` - код первого символа строки
- `abs()` - абсолютное значение числа


## Команды для сборки и запуска

### Сборка проекта

```bash
# Инициализация модуля 
go mod init config-parser

# Сборка проекта
go build -o config-parser
```

### Запуск тестов

```bash
# Запуск тестов с подробным выводом
go test -v
```
### Запуск программы
```bash
# Компиляция и запуск
go run main.go <input.conf> <output.toml>

# Использование скомпилированного бинарника
./config-parser <input.conf> <output.toml>
```
